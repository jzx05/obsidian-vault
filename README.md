# 🧠 知识库使用说明

## 📂 目录结构

| 目录 | 用途 | 何时写入 |
|------|------|----------|
| `00-Inbox/` | 临时收集箱 | 想法、链接、未分类内容先扔这里 |
| `10-Daily/` | 每日笔记（流水账） | 每天一篇，作为入口和索引 |
| `20-Papers/` | 论文库（按主题） | 读论文时按主题归档 |
| `30-Notes/` | 永久笔记（原子化） | 提炼出概念、方法、洞察 |
| `40-Projects/` | 进行中的项目 | 有明确目标和截止日期 |
| `50-Areas/` | 长期领域 | 持续关注的方向（如 WAM） |
| `90-Attachments/` | 附件（PDF/图片） | 论文原文、截图等 |
| `Excalidraw/` | 手绘图 | 思维导图、架构图 |
| `templates/` | 模板 | 新建笔记的起点 |

## 🔄 工作流

1. **遇到论文** → `20-Papers/<主题>/<标题>.md`（用 paper 模板）
2. **每天结束** → 在 `10-Daily/` 写当日笔记，链接今天读的论文/笔记
3. **提炼概念** → 从论文笔记里抽出独立概念到 `30-Notes/concepts/`
4. **想法零散** → 先丢 `00-Inbox/`，每周整理一次

## 📌 命名约定

- 论文：`<年份>-<简短标题>.md`，例：`2017-Attention-Is-All-You-Need.md`
- 日记：`YYYY-MM-DD.md`
- 笔记：用名词短语，例：`Self-Attention.md`、`Positional-Encoding.md`

## 🏷️ 状态标签

论文 frontmatter 里的 `status` 字段：
- `to-read` — 待读
- `reading` — 正在读
- `read` — 已读完
- `archived` — 不再关注

- 今日笔记：在 `10-Daily/` 找当天日期
