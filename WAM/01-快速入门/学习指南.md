## 🔑 核心概念速查表

### World Action Models (WAM)

```
定义: 通过联合预测视频和动作，继承物理世界先验的模型

关键特性:
- 视频预测 → 学习物理规律
- 动作预测 → 受物理约束
- 共享特征表示 → 知识转移
```

### 零样本泛化

```
含义: 无需在目标任务上训练，直接泛化到新任务

实现方式:
- 物理先验作为通用知识
- 视频预测约束动作合理性
- 跨域物理规律的通用性
```

### 跨具身迁移

```
含义: 从一个机器人/人类演示学习，转移到另一个机器人

机制:
- 物理规律具身无关
- 少量目标具身数据微调
- 保留零样本泛化能力
```

## 📊 关键数据点

### 模型规模

| 指标 | 数值 |
|------|------|
| 参数量 | 14B |
| 闭环频率 | 7Hz |
| 推理延迟 | ~140ms |

### 性能指标

| 指标 | DreamZero | 传统VLA | 提升 |
|------|-----------|---------|------|
| 零样本成功率 | 85% | 60% | +40% |
| 实时控制 | 7Hz | 1-2Hz | 3.5-7倍 |
| 跨具身迁移 | 双向 | 单向 | ✅ |
| 数据需求 | 仅DROID | 大规模 | ✅ |

### 适应数据需求

| 场景 | 数据量 | 效果 |
|------|--------|------|
| 零样本 | 0 | 强泛化 |
| 少样本适应 | 30分钟 | 新具身适应 |
| 完全微调 | 完整数据集 | 最优性能 |
