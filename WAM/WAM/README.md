## 💡 核心概念速查

### World Action Models (WAM)
通过**联合预测视频和动作**，让模型学习物理世界的规律

### 物理先验
视频预测学到的物理规律自动约束动作预测

### 零样本泛化
无需在目标任务上训练，直接泛化到新任务

### 跨具身迁移
物理规律在不同机器人间通用，支持双向迁移
